Minesweeper
===========

ノーマルなマインスイーパー

オブジェクト化不能なところを無理やり組み込んだせいでオブジェクト指向の根幹が揺らいでるが気にしない。<br />

Visual Studio 用準備
====================

+ http://www.xmission.com/~nate/glut.html より最新版xxxx-bin.zip をダウンロード
+ glut.h はC:\Program Files(x86)\Microsoft Visual Studio 1x.0\VC\include にGLフォルダを作成していれる
+ glut32.lib はC:\Program Files(x86)\Microsoft Visual Studio 1x.0\VC\lib にいれる
+ glut32.dll はwindowsが64bitなら C:\Windows\SysWOW64 へ32bitならC:\Windows\System32 へ入れる
